# XATA-CRUD

Install dependencies:

```bash
pnpm install
```

Run:

```bash
pnpm dev
```

## Description

1. Pull new notes from GitHub
2. see the full implementation of CRUD from deleteRequest folder
3. Integrate a serverless Database from https://xata.io/
